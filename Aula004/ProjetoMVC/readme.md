## Comandos 📝

Iniciar seu projeto com "init -y" e baixar express e mysql2

``
npm init -y && npm i express mysql2
``

Baixar TS, Tipos do node e o expre

npm i typescript ts-node-dev @types/node @types/express -D

Criar as configurçoes do **TS**

npx tsc --init
